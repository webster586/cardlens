# Recognition Pipeline

## Phase 1
- Mock Recognition für UI- und Persistenzfluss

## Phase 2
- Kartenbereich erkennen
- Perspektive entzerren
- Crop-Zonen definieren
- OCR auf Name / Set / Nummer
- Kandidatenlisten erzeugen

## Phase 3
- Mehrsprachige OCR-Pipeline
- Confidence Scoring
- Bildvergleich gegen Referenzbilder
- Holo-/Sleeve-spezifische Preprocessing-Pfade

## Confidence-Regel
- Default: immer bestätigen
- Automode: nur oberhalb definierter Schwelle
