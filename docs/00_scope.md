# Scope

## In Scope
- Windows Desktop App
- Mobile-first Bedienlogik innerhalb einer Desktop-App
- Fotoimport und später USB-Kamera
- persistente Sammlung
- Kandidatenliste mit manueller Bestätigung
- optionaler Automode bei hoher Confidence
- Preisdaten über austauschbare Adapter
- Export nach CSV, JSON, XLSX
- Logging, Crashlogs, Patchnotes

## Out of Scope für v1
- Nutzerkonten/Cloud
- Online-Sync
- Mobile App
- vollständige Bestandsbewertung ohne Bestätigung
- automatische Sleeve/Holo-Perfektion ohne Review
- eigene Bildtrainingspipeline
